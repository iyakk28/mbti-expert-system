Tentang Project

Ini adalah project Sistem Pakar untuk mendeteksi tipe kepribadian MBTI seseorang berdasarkan teori fungsi kognitif Carl Jung. Sistem menggunakan 60 pertanyaan dengan skala Likert 1-5 untuk mendapatkan hasil yang akurat.
